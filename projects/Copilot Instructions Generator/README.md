# 9898-MTG-Generators

A collection of generators and tools for running a Magic: The Gathering (MTG) Chaos RPG campaign. This repository provides scripts and utilities to create random events, encounters, and other content to enhance your MTG RPG experience.

## Features
- Random event generators
- Encounter and NPC generators
- Tools for campaign management
- Easily extensible for custom content

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/9898-MTG-Chaos-RPG.git
   ```
2. Navigate to the Generators directory:
   ```bash
   cd 9898-MTG-Chaos-RPG/9898-MTG-Generators
   ```
3. Follow instructions in each generator's script or module to run.

## Requirements
- Python 3.x (if using Python scripts)
- See individual generator scripts for additional dependencies

## Usage
Each generator is designed to be run independently. Refer to the documentation or comments within each script for specific usage instructions.
