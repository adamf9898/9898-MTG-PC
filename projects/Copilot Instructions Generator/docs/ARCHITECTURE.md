# Architecture Overview

## Structure

- `src/api/`: Scryfall API integration
- `src/core/`: Game state management
- `src/generators/`: Perchance-compatible generators
- `js/`: Main UI controller
- `styles/`: CSS and theming

## Design Decisions

- Modular ES6+ JavaScript
- Observer pattern for UI updates
- Accessible, responsive frontend
