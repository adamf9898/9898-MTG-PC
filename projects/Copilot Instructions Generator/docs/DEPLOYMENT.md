# Deployment

## Environment Setup

- Node.js 18+ required
- Install dependencies: `npm install`

## Deployment Steps

1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy the `dist/` or `build/` directory to your web server or GitHub Pages

## Environment Variables

- Configure API endpoints and secrets as needed

See [README.md](README.md) for setup instructions.
